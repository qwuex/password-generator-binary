# Password Generator with Binary Converter

Простой и удобный генератор безопасных паролей на Python, который умеет переводить результат в двоичный код (бинарный формат).

## ✨ Особенности / Фичи
- 🛠️ **Гибкая настройка:** можно выбирать, добавлять ли в пул генерации цифры и спецсимволы.
- 📊 **Масштабируемость:** генерация любого количества паролей заданной длины за один запуск.
- 🧠 **Бинарный конвертер:** каждый сгенерированный пароль автоматически переводится в двоичный код с помощью перевода символов через ASCII-таблицу (`bin(ord(char))`).

## 🛠️ Стек технологий
- **Язык:** Python 3.13.9
- **Модули:** `random`

## ❌Примичание
- Пароли создаются полностью случайно из-за чего запомнить тот или иной пароль может быть трудно.
